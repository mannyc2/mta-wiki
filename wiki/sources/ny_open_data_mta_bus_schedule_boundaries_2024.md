### [p001_b0001] Official dataset: MTA Bus Schedules: 2024
### [p001_b0002] Dataset identifier: udt9-hvjq
### [p001_b0003] Publisher: Metropolitan Transportation Authority
[p001_b0004] Dataset description: This dataset includes data for all bus schedules. It includes rows for each scheduled timepoint stop on a trip.
[p001_b0005] Schedule Day field: Represents the date for the beginning of schedule trips, in a MM/DD/YYYY format.
[p001_b0006] Trip Type field: The type of bus trip; 13 = Express service.
[p001_b0007] Origin field: Whether the stop is the origin (1) or the departure (0) terminal.
[p001_b0008] Service ID field: Name of service running for the day; the code includes the type of day (Weekday/Saturday/Sunday).
[p001_b0009] Route ID field: The name of the route.
[p001_b0010] Direction field: The N/S/E/W direction for the bus route. Many non-revenue trips have no defined direction.
[p001_b0011] Bundle field: Name of bundle in effect.
[p001_b0012] Bundle-boundary query: distinct schedule_date and bundle for 2024-06-29, 2024-06-30, 2024-08-31, and 2024-09-01, ordered by schedule_date and bundle.
